# US-1 Pools (Modern Reimagining)

A modern, design-forward reimagining of the US-1 Pools site with a multi-page layout, dropdown navigation, real project imagery, and SEO support. Built as a fast static site for easy deployment on Vercel.

## Pages

- Home: `index.html`
- About: `about.html`
- Pools Overview: `pools.html`
- Above Ground: `above-ground.html`
- In-Ground: `in-ground.html`
- Hot Tubs & Spas: `hot-tubs.html`
- Gallery: `gallery.html`
- Services: `services.html`
- Contact: `contact.html`
- FAQ: `faq.html`
- 404: `404.html`

## Assets

- Images: `assets/images`

## SEO & crawler support

- `sitemap.xml`

## Local preview

```bash
cd "/Volumes/Ext-code/GitHub Repos/us1pools"
python3 -m http.server 5173
```

Then open http://localhost:5173

## Deploy on Vercel

- Import the repository in Vercel.
- Framework preset: Other.
- Build command: none.
- Output directory: root.
